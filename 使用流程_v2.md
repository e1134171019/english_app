# 📖 英文練習 App - 使用手冊

**版本**：v3.0  
**最後更新**：2025-12-26

---

## 🚀 快速開始

### 啟動應用

1. **啟動服務器**（Windows）：
```powershell
cd d:\English_app
python -m http.server 8085
```

2. **開啟瀏覽器**：
```
http://localhost:8085/new_index.html
```

3. **驗證**：
   - 看到首頁 6 個功能卡片 ✅
   - 底部有 5 個導航按鈕 ✅
   - 無錯誤提示 ✅

## 📱 功能總覽

| 功能 | 說明 | 入口 |
|------|------|------|
| 📚 **單字練習** | 翻卡學習，支援發音與例句查詢 | 首頁 / 底部「單字」 |
| 🎧 **聽力測驗** | 聽音辨字，即時回饋 | 首頁 / 底部「聽力練習」 |
| 📝 **動詞三態** | 填空練習不規則動詞變化 | 首頁 / 底部「三態」 |
| ✨ **我的專屬** | 自訂單字清單，選擇練習模式 | 首頁 / 底部「我的專屬」 |
| 🔍 **互動例句** | 點擊例句單字即時查詢翻譯 | 所有練習模式背面 |

## 📚 功能 1：單字練習

### 操作流程

```
首頁 → 單字練習 → 選擇學制 → 選擇等級 → 開始 → 學習
```

### 詳細步驟

1. **進入練習**
   - 方式 A：首頁點擊「單字練習」卡片
   - 方式 B：底部導航點擊「單字」圖標

2. **選擇範圍**
   - **Step 1**：點選學制（國中 / 高中 / 進階）
   - **Step 2**：點選等級（例如：國二、高一）
   - 點擊「開始」按鈕

3. **學習卡片**
   - **卡片正面**：顯示中文翻譯
   - **點擊卡片**：翻轉到背面
   - **卡片背面**：顯示英文、音標、詞性、例句

4. **功能按鈕**
   | 按鈕 | 功能 |
   |------|------|
   | 🔊 | 播放單字發音 |
   | ⬅️ | 前一張卡片 |
   | ➡️ | 下一張卡片 |
   | ▶️ | 自動播放 |

### 互動例句（重點功能）

當卡片翻到背面時：

1. **識別可點擊單字**
   - 例句中的單字是**藍色** ✨
   - 滑鼠移上去會有底線

2. **點擊查詢**
   - 點擊任意例句單字
   - 彈出黑色半透明視窗（Tooltip）
   - 顯示：`單字: 中文翻譯`
   - 自動播放該單字發音 🔊

3. **自動隱藏**
   - 3 秒後自動消失
   - 或點擊其他位置關閉

## 🎧 功能 2：聽力測驗

### 操作流程

```
首頁 → 聽力練習 → 選擇等級 → 聽音 → 輸入 → 送出 → 下一題
```

### 詳細步驟

1. **進入測驗**
   - 方式 A：首頁點擊「聽力測驗」卡片
   - 方式 B：底部導航點擊「聽力練習」

2. **選擇範圍**（同單字練習）

3. **答題**
   - 🔊 自動播放單字發音
   - 📝 輸入框輸入聽到的單字
   - ✅ 點擊「送出」檢查答案
   - ⏭️ 或點擊「跳過」直接看答案

4. **查看結果**
   - ✅ **正確**：綠色提示
   - ❌ **錯誤**：顯示正確答案（紅色）
   - 點擊「下一題」繼續

### 小技巧

- **重聽**：點擊喇叭圖標可重複播放
- **不區分大小寫**：輸入 `apple` 或 `APPLE` 都正確

## 📝 功能 3：動詞三態

### 操作流程

```
首頁 → 動詞三態 → 選擇難度 → 填寫 → 檢查 → 上/下一個
```

### 詳細步驟

1. **進入練習**
   - 方式 A：首頁點擊「動詞三態」卡片
   - 方式 B：底部導航點擊「三態」

2. **選擇難度**
   - 國中 (JH)：基礎不規則動詞
   - 高中 (SH)：進階不規則動詞

3. **填寫答案**
   - 看到 **Base Form（原型）** 和中文
   - 填寫 **Past Tense（過去式）**
   - 填寫 **Past Participle（過去分詞）**

4. **驗證答案**
   - 點擊「檢查答案」
   - ✅ 正確：輸入框變綠色
   - ❌ 錯誤：顯示正確答案

5. **導航**
   - 「上一個」/ 「下一個」按鈕切換動詞

### 範例

| Base | Past | Past Participle |
|------|------|-----------------|
| go | went | gone |
| eat | ate | eaten |
| be | was/were | been |

## ✨ 功能 4：我的專屬（自訂題庫）

### 操作流程

```
首頁 → 我的專屬 → 輸入單字 → 選擇模式 → 開始練習
```

### 詳細步驟

#### A. 建立題庫

1. **進入管理**
   - 方式 A：首頁點擊「我的專屬」卡片
   - 方式 B：底部導航點擊「我的專屬」

2. **輸入單字**
   - 在文字框輸入單字列表
   - 支援格式：
     - 空格分隔：`apple banana orange`
     - 換行分隔：
       ```
       apple
       banana
       orange
       ```
   - 最多 100 個單字

3. **選擇練習模式**
   - 📚 **單字練習**：翻卡模式
   - 🎧 **聽力練習**：聽音辨字
   - 📝 **動詞三態**：僅動詞有效

4. **開始練習**
   - 點擊模式按鈕後自動建立題庫
   - 直接進入練習模式

#### B. 管理題庫

1. **查看題庫列表**
   - 顯示所有已建立的題庫
   - 顯示單字數量和建立時間

2. **使用已有題庫**
   - 點擊題庫卡片
   - 選擇練習模式開始

3. **刪除題庫**
   - 點擊 🗑️ 垃圾桶圖標
   - 確認刪除
   - 題庫永久移除

### 小技巧

- **快速建立**：直接輸入單字並選模式，自動儲存
- **重複使用**：題庫會保存，下次可直接選擇
- **驗證單字**：系統會自動檢查單字是否存在

## 🔧 常見問題

### Q1: 頁面無法載入

**症狀**：瀏覽器顯示 404 或空白

**解決方案**：
1. 確認目錄正確：`cd d:\English_app`
2. 確認服務器運行：`python -m http.server 8085`
3. 確認 URL：`http://localhost:8085/new_index.html`
4. 硬重新整理：`Ctrl + Shift + R`

---

### Q2: 卡片無法翻轉

**症狀**：點擊卡片沒反應

**解決方案**：
1. 確認點擊位置是**卡片中心**（不是按鈕）
2. 硬重新整理：`Ctrl + Shift + R`
3. 檢查瀏覽器 Console 是否有錯誤（F12）

---

### Q3: 聽不到發音

**症狀**：點擊喇叭圖標無聲音

**解決方案**：
1. 檢查電腦音量
2. 確認瀏覽器允許音訊播放
3. 第一次點擊後稍等片刻
4. 嘗試其他單字

---

### Q4: 互動例句沒反應

**症狀**：點擊例句單字沒有 Tooltip

**解決方案**：
1. 確認卡片已翻到**背面**
2. 確認點擊的是**藍色單字**
3. 硬重新整理：`Ctrl + Shift + R`

---

### Q5: 自訂題庫輸入無效

**症狀**：輸入單字後無法建立

**解決方案**：
1. 確認單字以**空格或換行**分隔
2. 確認單字為**英文單字**（系統會驗證）
3. 單字數量不超過 100 個
4. 選擇模式後會自動建立

## 💡 使用技巧

### 1. 快速切換模式

使用**底部導航**比返回首頁更快：
- 練習中 → 底部點擊其他模式 → 直接切換

### 2. 善用自訂題庫

- **課本單字**：輸入課本單字列表集中練習
- **錯題本**：記錄常錯單字，重複練習
- **主題學習**：建立主題單字庫（如：食物、動物）

### 3. 互動例句學習法

1. 先看中文（正面）
2. 嘗試回想英文
3. 翻卡驗證
4. **點擊例句生字**深度學習
5. 重複直到熟練

### 4. 聽力訓練技巧

- **第一遍**：盲聽，不看提示
- **第二遍**：如果不確定，再聽一次
- **檢查**：看到答案後，再聽一次確認發音
